Actually, there's a really easy way to do this:

A StringList nicely parses things into Name=Value pairs which is the same as
the ini file format.
StringList also has a CommaText property that flattens the list into a
single line.

So, you can flatten store an enitire obj's properties into a single line and
use it as a value.

eg.

procedure MyObj.WriteToStringList(StringList: TStringList);
Var
  ChildStringList: TStringList;
begin
  StringList.Clear;
  ChildStringList:= TStringList.Create;
  StringList.Values['Name']:= Self.Name;
  StringList.Values['IntProperty']:= IntToStr(Self.IntProperty);
  StringList.Values['BoolProperty']:= IntToStr( Ord(Self.BoolProperty) );

  ChildObj1.WriteToStringList(ChildStringList);
  StringList.Values['ChildObj1']:= ChildStringList.CommaText;

  ChildObj2.WriteToStringList(ChildStringList);
  StringList.Values['ChildObj2']:= ChildStringList.CommaText;

  ChildStringList.Free;
end;
