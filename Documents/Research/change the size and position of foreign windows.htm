...change the size and position of foreign windows?
Autor: Henning Huncke
Homepage: http://www.kraftda.de
[Write new comment]
[ Print tip ]  

Tip Rating (7):  
     


{
Diese function liefert True wenn das ändern der Fenstergröße erfolgreich war
und False wenn nicht. (True=changed with success False=changes couldn't be performed)

Handle = Fenster Handle von dem zu bearbeitendem Fenster
dHeight = Die neue Höhe des Fensters
dWidth = Die neue Breite des Fensters
ScreenCenter = Wenn True dann wird das Fenster auch direkt Zentriert wenn
False bleibts an der alten Position
}

function ChangeSize(Handle: Hwnd; dHeight, dWidth: Integer; ScreenCenter: Boolean): Boolean;
var
P: TRect;
begin
Result := False;
try
GetWindowRect(Handle, P); {Der TRect var die Positions daten des Fenster übergeben}
if ScreenCenter then
MoveWindow(Handle, (Screen.Width-dWidth) div 2, (Screen.Height-dHeight) div 2, dHeight, dWidth, True)
else
MoveWindow(Handle, P.Left, P.Top, dHeight, dWidth, True);
except
Result := False;
end;
Result := True;
end;
