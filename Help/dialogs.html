deLite Help [Dialogs]

Dialogs

This details the various special dialog commands that are in deLite

function dlg_Open(Title, Filter : String) : String

Usage:
         MyFile := dlg_Open('Select file to open', 'Text Files (*.txt)|*.txt')
Description:

Invokes the standard open dialog for your system, as used in most applications including this editor.

Returns

String value

function dlg_Save(Title, Filter, DefaultEXT : String) : String

Usage:
         MyFile := dlg_Save('Select file to open', 'Text Files (*.txt)|*.txt', '.txt')
Description:

Invokes the standard save dialog for your system, as used in most applications including this editor, set DefaultEXT to make the filename have that extension by default.

Returns

String value

Procedure dlg_Message(text : String)

Usage:
         dlg_Message('Hello World');
         or
         ShowMessage('Hello World');
Description:

Displays a simple one button "OK" message box, useful for displaying messages, can be used two ways. One is compliant with the dialog naming conventions, the other is direct from Pascal

Returns

Nothing

function dlg_Ask(const TipText : String) : Integer

Useage:
           MyInt := dlg_Ask('Do you wish to continue');
Description:

Use to show a custom ask dialog that allows a little more room than the standard windows dialog. Returns 0 for No, and 1 for Yes

Returns

Integer, 0 or 1

function dlg_Color(Title : String) : TColor

Useage:
            MyColor := dlg_Color('Select a color');
Description:

Allows you to ask user to select a color, and returns the selection as a TColor

Returns:

A TColor

function dlg_Text(Title : String) : String

Useage:
           txt := dlg_Text('Enter text');
Description:

A text entry dialog box with a lot more room than the standard windows dialog. The control used is actually a Memo, so 64k can be returned, and has scrollbars where needed.

Returns:

Text entered in the dialog, or '' if canceled

function dlg_Edit(const Title, CurrentText : String) : String

Useage:
           txt := dlg_Edit('Change Text', txt);
Description:

Same as above only the text box is set with CurrentText allowing you to edit text in the same fashion as the text entry dialog

Returns:

Text entered in the dialog, or '' if canceled

procedure dlg_UserPass(var username, password : String)

Useage:
           dlg_UserPass(Username, Password);
Description

Allows entry of a username and password and the password is obscured allowing more secure entry. The Microblog script makes use of this dialog to great effect. On Successful OK, the variables are set to the text entried in the username and password boxes. The password is obscured only while entered, the string returned is in no way encrypted, there are functions to do this for you.

Returns

On OK'ing the dialog Username, and Password variables are set
